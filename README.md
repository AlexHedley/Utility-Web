Web Utility App

Functions:

* HTML Encode/Decode
* HTML Escape/Unescape
* URL Encode
* URL Parser
* HEX to RGB
* SQL Builder (IN Clause)
* Guid
* JSON Pretty
* XML Pretty
* SQL Formatter
* Hidden Character Finder
* Binary
* MD5
* Diff
* Time Converter
* kb - mb - gb Converter
* luhn checker
* String Converter
* Notepad
* Ascii
* Picker Wheel
